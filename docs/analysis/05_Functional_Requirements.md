# 05. Functional Requirements (SRS Extract)

## FR-1: User Management
- JWT Authentication.
- Role-based Access Control (RBAC).

## FR-2: Catalog & Inventory
- Item-level tracking (Serial Numbers).
- Real-time availability status per branch.

## FR-3: Order Management
- Multi-item orders.
- Automated state transitions.
- Pricing calculation on-the-fly.

## FR-4: Auditing
- History of status changes for all `EquipmentItems`.
- Tracking of `STAFF` actions for accountability.
